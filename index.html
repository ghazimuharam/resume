Project list

Bert Transfer Learning to Classify Hoax Detection from text naration

Aplikasi untuk mendeteksi narasi hoax dibangun dengan menggunakan pendekatan machine learning. Model yang ada pada aplikasi ini dibangun dengan melakukan transfer learning dari base-model IndoBERT.

Simple Search Engine

Aplikasi ini dibangun untuk memudahkan pencarian pada data yang tidak terstruktur. Untuk melihat kedekatan query dengan document dilakukan pencarian cosine similarity berdasarkan kedua hal tersebut.

NLP Hoax Detection from text naration

Script yang ditulis dalam bahasa Python yang digunakan untuk membuat model machine learning dengan algoritma SVM yang ada pada library sklearn untuk melakukan klasifikasi narasi berita hoax.

Shopee Scraper

Script yang ditulis dalam bahasa Python yang bisa digunakan untuk melakukan scraping product pada situs e-commerce Shopee dengan bantuan Selenium WebDriver.

Cosine Measure

Aplikasi CLI yang di tulis dengan menggunakan bahasa pemrograman Python untuk mengurutkan ranking document berdasarkan query yang diberikan dengan menggunakan Cosine Measure/Cosine Similarity.

Berdebar Website

Berdebar (Berdiskusi & Beraksi) merupakan komunitas non-profit yang bergerak di bidang pengembangan kepemudaan atau Youth Development. Website ini dibangun dengan menggunakan Laravel 8.0.
